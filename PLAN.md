# PLAN.md — 신혼여행 가이드 웹사이트

## 1. 목표

바르셀로나(2박 3일) + 마요르카(2박 3일) 신혼여행 가이드북을 단일 `index.html`로 구현.  
아이폰 세로 화면 최적화 슬라이드 UI.

---

## 2. 파일 구조

```
index.html   ← 단일 파일 (HTML + CSS + JS 인라인)
```

외부 의존성: Google Fonts CDN (Playfair Display, Noto Sans KR)만 허용.

## 6. 슬라이드 데이터 구조 (JS 객체)

```js
{
  cities: ['barcelona', 'mallorca'],
  barcelona: {
    color: '#C4724A',
    accent: '#F0D9C8',
    groups: ['개요', 'DAY 1', 'DAY 2', 'DAY 3', '식당 총정리', '실용정보'],
    slides: [ /* 슬라이드 배열 */ ]
  },
  mallorca: { ... }
}
```

각 슬라이드 객체:
```js
{
  group: 'DAY 1',
  time: '오전',          // 오전|점심|오후|저녁|이른아침|이른저녁 (옵션)
  title: '파크 구엘',
  spots: [
    {
      name: '파크 구엘',
      mapUrl: 'https://www.google.com/maps/place/?q=place_id:...',
      rating: '4.4',
      reviewCount: '232,380',
      price: null,        // 식당만
      hours: '09:30~19:30',
      fee: '€10',
      reservation: '필수', // 필수|권장|불필요
      tips: [...],
      allergyBadge: null, // 마요르카 식당만: '최적'|'고지필요'
      parking: null       // 마요르카 스팟만
    }
  ]
}
```

## 7. 컴포넌트 명세

### 7-1. 탭바
- 높이 52px, 상단 고정
- 활성 탭: 메인 컬러 언더라인 2px
- 비활성 탭: #6B6B6B

### 7-2. 슬라이드 컨테이너
- `transform: translateX` 기반 전환
- `transition: 0.3s ease`
- 터치 스와이프: touchstart / touchmove / touchend

### 7-3. 슬라이드 카드
- 배경: #FFFFFF, border-radius: 16px
- box-shadow: 0 2px 12px rgba(0,0,0,0.08)
- 좌우 패딩: 20px

### 7-4. 구글맵 링크 행
- `<a href="..." target="_blank" rel="noopener noreferrer">`
- 🗺️ + 장소명 전체가 탭 영역
- 최소 높이 44px

### 7-5. 배지 컴포넌트
- 예약: 🔴 필수 / 🟡 권장 / ✅ 불필요
- 주차 난이도: 🔴 매우 어려움 / ⚠️ 보통 / ✅ 쉬움
- 새우 알러지: ✅ 최적 / ⚠️ 고지필요

### 7-6. 도트 인디케이터
- 현재 슬라이드 그룹 + 현재 슬라이드 위치 표시
- 활성 도트: 메인 컬러, 비활성: #D0D0D0

### 7-7. 토스트 (마요르카 새우 알러지)
- 하단에서 올라옴, 2초 후 자동 사라짐
- 클립보드 복사 후 표시

## 8. 구현 순서

1. **HTML 뼈대** — 탭바, 슬라이드 컨테이너, 도트 인디케이터
2. **CSS** — iOS Safe Area, 폰트, 컬러 변수, 컴포넌트 스타일
3. **데이터 레이어** — 바르셀로나/마요르카 슬라이드 데이터 JS 객체
4. **렌더 함수** — 슬라이드 HTML 동적 생성
5. **인터랙션** — 탭 전환, 스와이프, 도트 업데이트
6. **특수 기능** — 새우 알러지 클립보드 복사 + 토스트
